## 🚀 Featured Projects

🔹 **[Subscription Tracker (Python + MySQL CLI)](https://github.com/ranjanvrma/subscription-tracker)**  
A command-line based application to manage and track personal subscriptions using a structured database system.  
- Designed relational database schema in MySQL  
- Implemented CRUD operations for subscription management  
- Integrated Python with MySQL for backend functionality  
- Focused on clean CLI interaction and data handling  

🔹 **[Credit Card Fraud Detection (Machine Learning)](https://github.com/ranjanvrma/cc-fraud)**  
A machine learning project to identify fraudulent transactions using classification techniques.  
- Performed data preprocessing and handled imbalanced dataset  
- Applied classification algorithms (Logistic Regression, etc.)

- Focused on minimizing false negatives (fraud detection priority)
